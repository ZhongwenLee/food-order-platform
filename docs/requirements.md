# 外卖点餐平台需求文档

## 1. 项目背景
本项目是一个面向用户、商家和管理员的点餐平台，支持浏览商家、浏览菜品、购物车、下单和订单查询。

## 2. 项目目标
- 验证核心点餐流程是否可用
- 验证商家接单流程是否顺畅
- 验证管理员基础管理能力

## 3. 角色说明
### 用户
浏览商家和菜品、加入购物车、下单、查询订单

### 商家
管理菜品、查看订单、处理订单状态

### 管理员
管理用户、商家和基础平台数据

## 4. 功能清单
### 4.1 MVP必做功能
#### 用户端
- 注册/登录
- 浏览商家
- 浏览菜品
- 加入购物车
- 提交订单
- 查询订单

#### 商家端
- 登录
- 菜品管理
- 订单查看
- 订单状态处理

#### 管理员端
- 登录
- 用户管理
- 商家管理
- 订单查看

### 4.2 后续增强功能
- 支付
- 优惠券
- 评价
- 退款
- 配送跟踪
- 数据报表

## 5. 角色权限说明
### 用户
仅可操作自己的购物车和订单

### 商家
仅可管理自己的店铺菜品和订单

### 管理员
可管理平台用户、商家和基础配置

## 6. MVP范围说明
MVP 只实现核心闭环：
浏览商家 -> 浏览菜品 -> 加入购物车 -> 下单 -> 商家处理订单 -> 用户查询订单

暂不包含：
支付、优惠券、评价、退款、配送等功能

## 7. 业务流程
1. 用户浏览商家
2. 用户浏览菜品
3. 用户下单
4. 商家接单
5. 用户查询订单状态
